# Taximeter Simulator

React Native (Expo + TypeScript) で作る、タクシーメーター風シミュレーターです。

## 現在の実装

- 横向き（ランドスケープ）前提のメーターUI
- `運転開始 / 停止` ボタン
- 位置情報のリアルタイム取得
- 走行時間・走行距離のリアルタイム表示
- 東京 / 大阪 の料金プリセット切替
- 距離加算モード / 低速時間加算モードの切替表示

## セットアップ

```bash
npm install
npm run start
```

Android 端末で開く場合は、Expo Go を利用して QR から起動できます。

## APK / AAB ビルド（EAS）

- 検証用APK:
  - `eas build -p android --profile preview`
- 公開用AAB:
  - `eas build -p android --profile production`

## 料金体系（このアプリ内の実装）

### 東京

- 初乗り: 500円（1.0kmまで）
- 距離加算: 255mごとに +100円
- 低速時間加算: 時速10km以下のとき90秒ごとに +100円

### 大阪

- 初乗り: 600円（1.3kmまで）
- 距離加算: 260mごとに +100円
- 低速時間加算: 時速10km以下のとき95秒ごとに +100円

### 計算ロジック

- 時速10km以下なら「時間加算モード」で、規定秒数ごとに料金加算
- 時速10km超なら「距離加算モード」で、規定距離ごとに料金加算
- モードは位置情報更新ごとに判定し、料金をリアルタイム更新
